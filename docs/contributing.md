# Contributing

## Welcome

Contributions to DeepClaw UI are welcome! This document provides guidelines.

## Getting Started

1. Clone the repository
2. Install dependencies (if any)
3. Make your changes
4. Test locally
5. Submit a pull request

## Code Style

### JavaScript

- Use JavaScript ES6+ features
- Use descriptive variable names
- Comment complex logic

### HTML/CSS

- Use semantic HTML
- Follow existing CSS structure (custom properties)
- Test responsive behavior

## Testing

### Manual Testing

```bash
# Start server
node deepclaw-ui.js

# Test in browser
http://localhost:1234
```

### API Testing

```bash
# Check status
curl http://localhost:1234/api/status

# List sessions
curl http://localhost:1234/api/sessions
```

## Reporting Issues

Include:
1. Steps to reproduce
2. Expected behavior
3. Actual behavior
4. Environment details

## Feature Requests

1. Describe the feature
2. Explain use cases
3. Suggest implementation approach

---

*Thank you for contributing!*
